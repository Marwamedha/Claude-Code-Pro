# Migration and Analysis With ClAUDE Code


**Migration and Analysis with Claude Code** is a Python project built to assist with migrating data and generating visualizations using both code and AI workflows.

This README documents how to install, run, and use the visualization features provided in this project.

---

## 🚀 Project Overview

This project is designed to help you:

- Perform migration-related data transformations
- Generate visualizations using Python and AI tools
- Use Claude Code (or other AI assistants) to produce diagrams and charts
- Analyze project code structure and database relationships
- Explore metrics and trends visually

---

## 📦 Requirements

Make sure you have the following installed:

- Python 3.10+
- `uv` package manager (as used in this project)
- Dependencies listed in `pyproject.toml`

---

## 📥 Install Dependencies

In your project root directory run:

```bash
uv install
